# CSSwitch v0.3.3 — 跨平台桌面包与远程 Linux Helper

这是 CSSwitch 第一个完整跨平台构建版本：Windows x64、Windows arm64、macOS Apple Silicon 都已经进入同一套 GitHub Actions 发布矩阵；远程 Linux helper 也同步提供 x86_64 与 aarch64 两个架构。换句话说，这版不是临时补包，而是把后续持续发版要用的基础设施搭起来了。

本次重新发布额外修复了远程 SSH 密码登录：用户可以在远程服务器表单里输入密码并选择是否记住；首次连接会用临时密码完成 SSH 认证，连接成功后才按选择写入凭据存储，密码不会写进远程 profile JSON。

## 下载哪个文件

- **Windows x64**：下载 `CSSwitch_*_x64-setup.exe`，适用于绝大多数 Intel / AMD Windows 设备。
- **Windows arm64**：下载 `CSSwitch_*_arm64-setup.exe`，适用于 Windows on Arm 设备。
- **macOS Apple Silicon**：下载 `CSSwitch_*.dmg`，适用于 M 系列 Mac。
- **远程 Linux helper**：`csswitch-helper-linux-x86_64` / `csswitch-helper-linux-aarch64` 是远程服务器模式使用的辅助二进制，不是桌面安装包。普通桌面用户不用单独下载它；桌面 app 会按需准备。

## 本次新增

- **Windows 双架构安装包**：CI 现在同时构建 x64 与 arm64 NSIS 安装器，Windows 用户不再需要自己从源码打包。
- **macOS arm64 DMG**：继续沿用当前 macOS Apple Silicon 发布路线，保持与原有桌面方案一致。
- **远程 Linux helper 双架构**：新增 x86_64 与 aarch64 静态 helper，服务于“本机桌面管理远程 Linux Science 沙箱”的模式。
- **远程服务器管理闭环**：补齐 SSH 认证、helper 自动准备、远程代理启动、远程沙箱启动、状态检测、日志查看与停止流程。
- **远程密码登录修复**：密码认证支持临时使用与可选记住；系统凭据存储不可用时会回退到本地加密文件，仍避免把明文密码写进 `remote-hosts.json`。
- **可复用发版矩阵**：`main` 分支用于验证全平台构建，`v*` tag 用于触发正式 Release，安装包和 helper 会自动挂载到 Release 页面。

## 主要功能回顾

- 在 Claude Science 中使用自己的第三方模型 API Key，不让推理走 Anthropic 推理服务。
- 支持 DeepSeek、通义千问、智谱 GLM、Kimi、MiniMax、小米 MiMo、硅基流动、OpenRouter，以及自定义 OpenAI / Anthropic 兼容端点。
- 支持多 profile：同一个服务商可以保存多套 key、base_url 与模型配置，并一键切换。
- Science 顶部模型选择器显示真实模型名，例如 `glm-5.2`，不是笼统的 `claude` / `opus`。
- 本地登录状态在隔离沙箱中生成；真实 Science 数据目录、真实 OAuth token 与 8765 端口继续按项目铁律保护。
- 桌面 app 是正常窗口进程管家，负责启动代理、准备沙箱、切换官方 / 第三方模式、展示状态灯和自检信息。

## 项目关系与致谢

CSSwitch 的命名和产品形态受 [CC Switch](https://github.com/farion1231/cc-switch) 启发。CC Switch 是一个 MIT 许可的 Claude Code API 服务商切换工具，CSSwitch 借鉴了它“服务商 / profile 切换工具”的产品思路，并在 README 中保留了致谢。

需要说明清楚的是：CSSwitch 是面向 Claude Science 的独立项目，不隶属 CC Switch，也不受 CC Switch 或 Anthropic 背书。当前发布包里的桌面 app、Python 代理、远程 helper、沙箱启动和本地登录实现均来自 CSSwitch 本仓库，没有打包 CC Switch 的二进制。如果未来引入 CC Switch 的 MIT 代码模块，会在许可证和说明中明确标注。

## 已知限制

- Windows 安装器当前未代码签名，首次安装可能触发 SmartScreen；请只从本仓库 Release 下载。
- macOS DMG 当前为 ad-hoc 签名、未 Apple notarization，首次打开需要右键“打开”或在系统设置的隐私与安全性中放行。
- macOS 目前只发布 Apple Silicon arm64 包，暂不发布 Intel / Universal 包。
- `csswitch-helper-linux-*` 是远程模式组件，不是 Linux 桌面版。
- 使用本地登录状态时，Anthropic 托管的远程 MCP 服务可能不可用；CSSwitch 内置或本地配置的工具不受此限制。
